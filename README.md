# EIGRP Dynamic Routing Project (Cisco Packet Tracer)

## 📌 Project Overview
This project demonstrates the implementation of EIGRP (Enhanced Interior Gateway Routing Protocol) using Cisco Packet Tracer. It is used to enable dynamic routing between multiple routers and ensure efficient communication across different networks.

## 🏗️ Network Design Features
- EIGRP dynamic routing configuration
- Multiple router connectivity
- IP addressing for different networks
- Automatic route sharing between routers
- Verification of network reachability using ping and routing tables

## 🔧 Technologies Used
- Cisco Packet Tracer
- EIGRP (Enhanced Interior Gateway Routing Protocol)
- IP Addressing & Subnetting
- Router Configuration
- CLI (Command Line Interface)

## 🎯 Objectives
- To understand dynamic routing using EIGRP
- To establish communication between multiple networks
- To learn how routers exchange routing information automatically
- To improve troubleshooting skills using routing commands

## 🧠 Learning Outcomes
- Hands-on experience with EIGRP configuration
- Understanding of routing tables and neighbor relationships
- Improved knowledge of dynamic routing protocols
- Practical networking troubleshooting skills
